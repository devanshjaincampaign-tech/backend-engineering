# Backend Engineering

A structured backend engineering repository focused on Node.js, REST APIs, databases, authentication, and scalable backend architecture.

---

## Overview

This repository documents my backend engineering journey through practical implementation, experimentation, and project-based learning.

The goal of this repository is to build strong backend fundamentals by understanding how backend systems work internally rather than relying only on abstractions and frameworks.

---

## Topics Covered

### Core Backend Fundamentals
- Raw Node.js HTTP servers
- Request/Response lifecycle
- Routing
- HTTP methods and status codes
- Headers and content types
- File system operations

### Backend Development
- REST APIs
- Express.js
- Middleware
- Error handling
- Authentication & Authorization
- JWT & Sessions
- Password hashing

### Databases
- MongoDB
- PostgreSQL
- Database integration
- CRUD operations
- Query optimization basics

### Advanced Concepts
- WebSockets
- Caching
- Rate limiting
- API security
- Deployment
- Scalable backend architecture
- System design fundamentals
